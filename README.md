# Smart Inventory System

A full-stack web application designed to help small and medium businesses gain better visibility into their inventory by tracking stock levels, sales activity, and identifying fast-moving and slow-moving items.
This project provides a simple, tech-enabled solution to analyze inventory movement and support better stocking decisions.

---

## üí° Solution Overview

The Smart Inventory System:
- Maintains inventory records
- Tracks sales transactions
- Calculates sales velocity over the last 30 days
- Classifies items as **Fast-moving** or **Slow-moving**
- Updates stock levels in real time

The system is intentionally lightweight and practical, focusing on **impact rather than complexity**.
